# Proyecto Semana 02 – Gestor de Trazabilidad Agrícola

## Dominio
Sistema de Trazabilidad Agrícola

## Descripción
Aplicación web desarrollada en JavaScript ES2023 que permite gestionar lotes agrícolas aplicando principios de programación funcional e inmutabilidad.

## Funcionalidades
- Crear, editar y eliminar lotes
- Marcar como activo/inactivo
- Filtros por estado, categoría y prioridad
- Búsqueda por texto
- Estadísticas dinámicas con reduce()
- Persistencia en LocalStorage

## Conceptos Aplicados
- Spread Operator
- Métodos de Array (map, filter, reduce, find)
- Inmutabilidad
- Manipulación del DOM
- Almacenamiento Local
